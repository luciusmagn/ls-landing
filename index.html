Hiisi

hiisi.app · Open for accounts

Hiisi is a language learning application built around structured translation. It translates a sentence, breaks the grammar down word by word, and turns the result into spaced-repetition flashcards. Every translation names three languages: the language you translate from, the language you translate to, and the language the explanations are written in.

Structured translation Each translation arrives as structured data: the sentence, its translation, word forms, and notes.
Grammar breakdowns Word-by-word analysis with linked grammar terms, written in your explanation language.
Spaced repetition The SM-2 algorithm schedules each flashcard for review before it is forgotten.
Decks Cards are organized into decks. Decks can be shared and imported.
Starter decks Curated decks give a guided entry into a new language.

Hiisi is built by an educator. It serves self-learners as well as teachers, lecturers, and tutors.

ClankerMails

clankermails.com · Early access

ClankerMails gives a program its own mailbox. It accepts inbound mail for addresses at clankermails.com, stores messages and attachments, and hands them to your code over HTTP.

Hosted inboxes An address at clankermails.com without running mail infrastructure.
REST API Programs fetch incoming messages from a small HTTP API instead of speaking IMAP.
Webhooks A callback fires when mail arrives, so automation stays event-driven.
Scope Inbound mail for programs. No mail client, no sending suite, no consumer inbox.
