01 / Product

Hiisi

Automatic structured translation meets spaced repetition. Language learning that respects your time and actually works.

14 Languages

Translate between any pair of 14 supported languages -- including Finnish, Estonian, Czech, and others you will not find on most platforms.

Grammar Breakdowns

Every translation includes word-by-word analysis with part-of-speech tags, declension patterns, conjugation forms, and an interactive grammar term wiki.

Spaced Repetition

Save translations to custom decks and review them as flashcards. The SM-2 algorithm schedules reviews at optimal intervals for long-term retention.

Shareable Decks

Create decks, share them with a link, and let others import your curated vocabulary. Also available for schools and institutions.

02 / Approach

Off the Beaten Path

We are distrustful of mainstream defaults and willing to venture where fewer developers go. We choose tools that are rewarding to work with, not just popular.

()

Common Lisp

We chose Common Lisp because it is cool, because its technical aspects are genuinely compelling, and because the REPL-driven workflow is unmatched. SBCL gives us native compilation and a live development experience.

fn

Rust

Rust where it fits: authentication, payments, email, background tasks. A language with real guarantees and a type system that earns its keep. Rocket framework, Diesel ORM, zero hand-waving.

hx

HTMX

Server-rendered HTML with surgical DOM updates. No JavaScript framework, no virtual DOM, no bundle size anxiety. Hypermedia as the engine of application state.

db

SQLite

Simple, fast, zero-config. One file, shared between both backends. No unnecessary infrastructure complexity. Sometimes boring technology is the right technology.

04 / About

Lambda Symbolics OÜ

Lambda Symbolics is a one-person company registered in Estonia, led by Lukáš Hozda. We build things with interesting technologies and have fun with the craftsmanship.

We are distrustful of mainstream choices and willing to venture off the beaten path. We use Common Lisp because it is genuinely cool and technically rewarding. We use Rust because it makes real promises and keeps them. We pick tools that make us excited to write code, not tools that everyone else already picked.

The goal is simple: build fewer things, build them well, and enjoy the process of making them. Every technology choice is deliberate. Every line of code is written with care. No frameworks-of-the-week, no hype-driven development.

Interesting Tools, Better Results

Common Lisp in production. Dual-backend architecture. We pick technologies that are rewarding to work with, and the results speak for themselves.

Boring Where It Counts

SQLite over Postgres clusters. Server rendering over SPA frameworks. Simplicity is a feature. Not every choice needs to be exotic.

Craftsmanship Over Process

Small team, strong opinions, deep care. We would rather spend time writing good code than writing tickets about writing code.
