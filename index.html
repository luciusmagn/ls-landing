Software

Hiisi

hiisi.app

Language learning through structured translation. Hiisi translates a sentence, explains its grammar word by word, and turns the result into spaced-repetition flashcards. Accounts are open.

ClankerMails

clankermails.com

Hosted mailboxes for programs. Inbound mail becomes a small REST API and webhooks, so software can receive email without running mail infrastructure. In early access.

Teaching

Live online courses taught by Lukáš Hozda, a university lecturer with prior commercial Rust training. The first course runs ten weeks: twenty live sessions in which every student builds a Game Boy emulator in Rust. Dates are to be announced.
