03 Technical System

Direct systems, explicit state

The company stack favors inspectability, low ceremony, and long-lived infrastructure over fashion-driven complexity.

Runtime arrangement

Hiisi production system

Interface

Server-rendered HTML, HTMX interactions, static assets.

Application core

Common Lisp for translation logic, flashcards, deck workflows, and UI composition.

Control plane

Rust for authentication, billing, email, and long-running background work.

State

SQLite for the primary datastore, Redis for transient data and fast lookups.
